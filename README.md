# Book Sales Analytics Project (SQL + Python)
SQL + Python analytics project analysing 10,000+ books and sales records to explore relationships between ratings, authors, and sales trends.

## Objectives
- Build a relational database using SQLite
- Generate synthetic sales data using Python
- Perform SQL-based analysis on sales and book metadata
- Visualise trends and relationships using Python

## Tech Stack
- Python
- SQLite (SQL)
- Pandas
- Matplotlib / Seaborn

## Key Features
- Joined sales and book metadata tables
- Aggregated sales by author, rating, and time period
- Analysed relationship between ratings and sales
- Identified sales trends over time
- Visualised insights using Python

## Insights Explored
- Do higher-rated books sell more?
- Which authors generate the highest sales?
- How do sales change over time?
